Special mentions

EU FP7 NEMESYS

The main goal of NEMESYS is the research and development of novel security technologies for seamless service provisioning in the smart mobile ecosystem. Mobile and smartphone security is a fast moving field. To this end, NEMESYS gathers and analyses information about the nature of cyber attacks targeting smart mobile devices, so that appropriate countermeasures can be taken to combat them.

Yara

YARA is a tool aimed at (but not limited to) helping malware researchers to identify and classify malware samples. With YARA you can create descriptions of malware families (or whatever you want to describe) based on textual or binary patterns. Each description, a.k.a rule, consists of a set of strings and a boolean expression which determine its logic.

Uptodown

Uptodown is an international download site for desktop software and mobile apps. It was created on December 20, 2002, and is based in Málaga, Spain. Its catalogue currently includes more than 30,000 applications for the main computer operating systems, divided into themed categories and stored on Uptodown’s own servers. Each application has its own information page written by the Uptodown editorial team and complemented with user opinions.

MI3 Security

Mi3 Security is the leader in intelligence-driven mobile app security. The company’s SaaS solutions offer unmatched, comprehensive analysis of Security and Privacy risks for iOS and Android applications. Mi3Security employees a patent-pending risk analysis platform that continually assesses app contents, the context surrounding the app, and code intentions to predict risk and quickly declare an app In or Out™ of compliance.
